# GR-MANGO

## Spec
 - CPU: RZ/A2M(R7S921058VCBG), Cortex-A9
 - RAM: 4MB, DRAM:16MB
 - ROM: 16MB

## LED
 - P0_1: Green
 - P0_3: Yellow
 - P0_5: Orange
 - P8_2: Red

https://www.renesas.com/jp/ja/software-tool/rza2m-freertos-software-package#document

# Boot Mode 
ブートモード:6 16MByte OctaFLASHから起動
MD_BOOT0: LOW固定
MD_BOOT1: HI固定
MD_BOOT2: HI固定
